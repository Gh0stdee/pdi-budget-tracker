# pdi-budget-tracker

## Setup
```
git clone https://github.com/Gh0stdee/pdi-budget-tracker.git
cd ./pdi-budget-tracker
echo . > .env
uv sync
```

### Add database file name in .env

```
DATABASE = "sqlite:///some_sql_file_name.db"
```
